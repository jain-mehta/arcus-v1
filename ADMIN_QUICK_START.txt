Q: How do I add another admin user?
A: Run this SQL:
   INSERT INTO user_roles (user_id, role_id)
   SELECT u.id, r.id FROM users u, roles r
   WHERE u.email = 'newadmin@company.com' AND r.name = 'admin';

Q: How do I remove admin access?
A: Run this SQL:
   DELETE FROM user_roles 
   WHERE user_id = (SELECT id FROM users WHERE email = 'user@company.com')
   AND role_id = (SELECT id FROM roles WHERE name = 'admin');

Q: How do I verify admin setup worked?
A: Run this SQL:
   SELECT u.email, r.name FROM users u
   JOIN user_roles ur ON u.id = ur.user_id
   JOIN roles r ON ur.role_id = r.id
   WHERE u.email = 'admin@arcus.local';

Q: Can I customize what admin can access?
A: Yes, see ADMIN_PERMISSIONS_GUIDE.md for how to modify permissions.

Q: What if setup fails?
A: Check ADMIN_PERMISSIONS_GUIDE.md > Common Issues section.
